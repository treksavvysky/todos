# Vision Statement: Task Manager

## 🎯 The Mission
To provide a **frictionless, local-first productivity environment** that empowers users to manage their lives with categorical clarity. This tool is designed for those who value speed, privacy, and the ability to distinguish between high-level areas of responsibility (**Scopes**) and specific, actionable outcomes (**Projects**).

## 💎 Core Pillars

### 1. Local-First Performance
The application is built on the belief that managing tasks should be instantaneous. By utilizing a local SQLite database and Next.js Turbopack, we ensure zero-latency interactions. Your data lives where you do, making it fast, portable, and private.

### 2. Categorical Clarity
Most tools blur the line between "context" and "goal." Our vision maintains a strict but flexible hierarchy:
- **Scopes:** The broad containers of life (Work, Personal, Health).
- **Projects:** Time-bound, specific efforts within a Scope.
This separation allows for a "clean mind" approach to task management.

### 3. Frictionless Interaction
The interface should never be an obstacle. Through a **Keyboard-First** design (shortcuts for everything) and **Bulk Operations**, we cater to power users who need to process large volumes of information without ever leaving the "flow state."

### 4. Adaptive & Modern Aesthetics
A tool you use every day should be beautiful and adaptive. Using Tailwind CSS v4 and a refined Light/Dark theme, the Task Manager provides a polished, modern experience that feels native on both desktop and mobile.

## 🌈 The Future Experience
We imagine a workspace where:
- **Data is truly yours:** Easy exports and local backups.
- **Insights are automatic:** Understanding where your time goes across Scopes.
- **Focus is the default:** A "Zen Mode" or Kanban view to eliminate noise and highlight only what matters *now*.

---

*This vision serves as the North Star for all technical and functional decisions in this repository.*
